# SES-Net: Semantic-Edge Synergistic Network for Industrial Small Defect Detection
The `block`.py serves as the coordinating hub that governs the interaction between the two modules. The perfect attention maps required by CSSG-Loss, ACFEE, and CSSG are located in the `module` folder. These two modules are designed to be plug-and-play. The baseline model adopted in this paper is RTDETR-R18.
